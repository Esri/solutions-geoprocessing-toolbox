## Incident Analysis Tools

These tools help you discover spatial patterns and trends in significant activity in your area of interest.

* Cluster Analysis
* Count Incidents By LOC
* Find Percent Change
* Hot Spots By Area
* Incident Density
* Incident Hot Spots
* Incident Table To Point
